Project Overview

Teslim’s Design System is a personal framework I created to streamline my design and development workflow. I realized that recreating components, variables, styles etc. from scratch for each project was inefficient, so I built a reusable system of components, styles, and patterns to speed up future projects. This system was developed entirely by me over the course of one month.

VIEW DESIGN SYSTEM

The Problem

Rebuilding the same components for each personal project became repetitive and time-consuming. This slowed down my workflow, reduced consistency, and limited the polish of my personal work.

Research & Insights

I studied popular design systems like Material Design, Shade UI and UI Prep Design System to understand how reusable patterns improve efficiency. The key insight was that having predefined components and consistent styles reduces decision fatigue and accelerates design and build processes.

Design Strategy

My approach was to create a flexible yet opinionated system that could adapt to different project needs. I focused on atomic design principles, building from colors and typography up to complex layouts. Accessibility, scalability, and clarity were top priorities.

Process

The system was built from the ground up, starting with a variable-based color palette and typography scale. I then designed reusable UI components such as buttons, forms, navigation, and cards. Finally, I documented usage guidelines and created responsive layouts to test system flexibility across devices.

Final System

The final design system includes typography scales, preset spacing variables, color themes, UI components, and documentation. It significantly reduces time spent on repetitive work and ensures consistency across all my personal projects.

Outcome & Reflection

Creating this design system was a pivotal step in my growth as a designer. It gave me the opportunity to translate theory into practice, experiment with variables for consistency, and build components that emphasize reusability and scalability. Beyond just a project, it became a framework that shaped how I now approach design with structure, clarity, and efficiency.
